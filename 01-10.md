## [Level 7 - he went to the old school](https://notpron.com/notpron/sdrawkcab/tieman.htm)

First things first when looking at the url we can see this weird word "sdrawkcab", if you just noticed its backwards backwards, changing it in the url opens up a new page just saying "no, just name it", I guess this could also be a hint towards the solution.

Then at the bottom in the hints and rules section it tells us "From now on almost all levels require you to have a look at the source code", so we'll do that.

Looking at the end of it we can see the comment `<!--what candy wrapper is it?-->`, so from that we can assume the image that we are seeing contains a wrapper from some candy, after some investigation I figured it is a "Twix".

Since there doesn't seem to be any clickable stuff in the page I'll try changing the url instead:

+ In the url I figured the part that says "tieman" is "name it" backwards, so I put in there "twix" instead.
+ Now the page says "wenoot", "too new" backwards, so I searched in Google the original name of "Twix" and I got "Raider".
+ I put it in the url and it shows "dnuorayawrehto", "other way around" backwards, so finally I put "rediar" and I get to the next level.

**Bonus**: originally I put "redair" and the page told me "Your spelling sucks!", I guess they were right.

## [Level 8 - say it loudly and proudly](https://notpron.com/notpron/sdrawkcab/rediar.htm)

At first glance there isn't something worth to look at in the page, we inspect the source code and see a custom tag hiding the existence of a file `<au dio="" src="../stuff/mus2.mp3">...`, if we try to reproduce its content there doesn't seem to have any audio at all, so we try and open the file directly in the browser `https://notpron.com/notpron/stuff/mus2.mp3`.

Opening it displays "No video with supported format and MIME type found", so instead we download it by right clicking the page and choosing "Save Page As...".

With this file we can try everything but it won't reproduce any sound, that's because if we see its file type (with an online tool or the `file` command from Linux) we get that this file its really an image, specifically a JPEG (this is also hinted by the capitalized letters JAY PACK, if you say them out loud it sounds like JPEG).

Finally we open the image and get the credentials, to get the form we click in the sound hole of the guitar (the hint also guides us towards trying to "re-format" the file).

## [Level 10 - he was impatient](https://notpron.com/notpron/wrong/roawr.htm)

The first thing we notice in this page is that the background noise has changed, and inspecting the page it doesn't seem to hold anything more of value aside from the comment `<!--%SystemRoot%\System32\sndrec32.exe can capture "what u hear"-->`, this may be useful (in the end it wasn't, just Audacity needed).

Then I inspect this audio in Audacity, my first instinct was to see the spectrogram but there wasn't anything in there and the audio itself doesn't resemble any speech.

So I go back to the source code and look for hints, the first thing I notice is a comment `<!-- quick quick, don't let me wait -->`, this seems to indicate that maybe we need to speed up the audio, another thing was the hint itself when pressing the ring in the image to open the form, it says "operator woman said it", this at least to me indicates that the credentials should be in a speech.

With this new information I go back to the audio, first I try to speed it up, I get to 8 times and it somewhat resembles a voice but not quite, then I just guessed the audio might be reversed, since I heard reversed speech before and it seemed similar, I did it and finally got the credentials with that.
